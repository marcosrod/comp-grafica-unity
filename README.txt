Tema: O Hangar Misterioso
Objetivo: Simular um ambiente de filmes de ação/aventura, onde se encontra um local com objetos misteriosos.
----------------------------//------------------------------------------------------------------------------------------------------------------//----------------------------
Formas de Interação:
	Mouse: É possível movimentar a câmera com o mouse nos sentidos X e Y.
	Teclado: É possível movimentar a câmera com o teclado, para os movimentos 
		de rotação e translação.

	OBS: Utilizando as duas formas de movimentação em conjunto (mouse e teclado) 
		é possível se movimentar pela cena e observar os objetos ao mesmo tempo.
----------------------------//------------------------------------------------------------------------------------------------------------------//----------------------------
Requisitos Adicionais/Opcionais Implementados:
	1. Movimentação da câmera com o mouse.
----------------------------//------------------------------------------------------------------------------------------------------------------//----------------------------
Informações Adicionas:

	Repositório para acessar o projeto:
		https://github.com/marcosrod/comp-grafica-unity

	A cena do projeto fica no caminho Assets/HQ Hangar Free/Scene/ExampleScene.unity

	1. Foi utilizado o tema Hangar no trabalho, os Assets foram importados através do link:
		https://assetstore.unity.com/packages/3d/environments/hq-hangar-free-212795

	2. Dentro da cena do projeto (ExampleScene), existem os objetos que já vieram importados dos Assets acima, e os objetos criados e editados para este trabalho, são eles:
		* Main Camera
		* Point Light
		* Spot Light
		* Espelho
		* CuboTransparente
		* Capsule

	3. Dentro da pasta Assets/Scripts, existe o script em C# criado para realizar a movimentação da câmera pelo mouse e teclado.
	4. Dentro da pasta HQ Hangar Free/Materials, existe o material "Transparente", criado e utilizado no CuboTransparente, para que o objeto ficasse transparente.
	5. Dentro da pasta HQ Hangar Free/Textures/Concrete_floor, existe a textura Concrete_floor_smooth_Normal.png, utilizada para realizar o normal map.
